# desafio-dashboard-dio
Desafio do bootcamp da TOTVS


Este projeto consiste em uma planilha automatizada no Excel (.xlsx) que simula vendas de uma distribuidora de ventiladores, o arquivo possui uma divisão entre a camada de dados e uma interface visual(dashboard).

##  Estrutura do Arquivo

O arquivo está organizado em duas abas principais:

 **Dashboard:**
   Blocos destacados no topo com Faturamento Total, Unidades Vendidas e Lucro Líquido acumulado

   **Gráficos Integrados:** 
    Gráfico de Pizza exibindo o market share de faturamento por Canal de Venda e um gráfico de barras Indicativo mostrando a performance regional de vendas

## Dados
   A base de dados contem 150 registros, os atributos são: ID do Pedido, Data da Venda, Canal (E-commerce, Loja Física, etc), Região, Modelo do Ventilador, Preço, Quantidade, Faturamento, Custo e Margem Operacional
    
   **AVISO:** Todos os dados usados foram criados por IA e portanto, são simulados
  

## Tecnologias e Técnicas Utilizadas


- **Design de Interface (UI/UX no Excel):** 
  - Formúlas dentro do excel especializada em dashboards
  - Paleta de cores corporativa baseada em tons Deep Teal (Azul-Petróleo) e cinzas suaves.
  - Formatação condicional implícita via preenchimentos zebrados alternados para legibilidade.
  - Formatação de moedas e alinhamentos numéricos padronizados à direita
